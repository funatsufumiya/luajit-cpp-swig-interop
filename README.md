# LuaJIT / C++ interop using SWIG

This is tiny example interop between LuaJIT and C++

## Usage

(for example on mac)

```bash
$ cmake -B build -GXcode -DCMAKE_BUILD_TYPE=Debug
$ cmake --build build --parallel 8 --config Debug
$ ./build/Debug/luajit-cpp-swig-interop
```
